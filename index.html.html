Le Concept

L'OBJECTIF :
ZÉRO JETON

Chaque joueur démarre avec 15 jetons. Pour gagner, vous devez tous les défausser en validant vos analyses football devant vos adversaires.

Bonne analyse ? Vous défaussez vos jetons. Erreur face à la VAR ? Vous les gardez et votre tour s'arrête. Simple. Implacable.

La base de données est mise à jour régulièrement. Nouveaux transferts, nouvelles saisons — VAR Club ne vieillit jamais.

VOTRE MODE

01
🌟
L'Identité

Tire une carte club. Annonce la ville, le stade, le surnom avant la révélation. Chaque bonne réponse = jetons défaussés.

Le Jeune Passionné
02
🛡️
Le Duel VAR

Tire une carte Club + une carte Question. Fais ton annonce à voix haute. Si la VAR confirme, tes jetons tombent. Sinon, tu attends.

Confrontation directe
03
🔥
Le Mode Expert

Data brute, transferts, statistiques pointues. Un sprint final pour vider son stock de jetons le plus vite possible. Seuls les pros survivent.
