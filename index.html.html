Le Concept

L'OBJECTIF :
ZÉRO JETON

Chaque joueur démarre avec 15 jetons. Pour gagner, vous devez tous les défausser en validant vos analyses football devant vos adversaires.

Bonne analyse ? Vous défaussez vos jetons. Erreur face à la VAR ? Vous les gardez et votre tour s'arrête. Simple. Implacable.

La base de données est mise à jour régulièrement. Nouveaux transferts, nouvelles saisons — VAR Club ne vieillit jamais.

La Règle d'or

LA VAR A LE
DERNIER MOT

L'application est votre assistant, pas votre maître. Les vrais arbitres, c'est vous. Le foot, c'est avant tout des débats passionnés autour de la table.

I
Débattez librement

L'arbitrage humain d'abord. Si tout le monde est d'accord, le point est validé sans application.

II
Un doute ? VAR !

Un adversaire crie "VAR !" — on vérifie sur l'interface. Le recours à la vidéo est facultatif.

III
Verdict positif

La VAR confirme votre analyse. Vous défaussez vos jetons. La victoire se rapproche.

IV
Hors-jeu

Erreur. Vous gardez vos jetons et attendez le prochain tour. La VAR est implacable.
